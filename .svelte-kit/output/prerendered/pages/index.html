# Redirecting

You might want the home page to be the first page of your documentation. If this is the case, rename this file to index.svelte and replace the content of this file with the following:

svelte 
<script context="module">
  export const prerender = true;

  /** @type {import("@sveltejs/kit").Load} */
  export function load() {
    return {
      status: 307,
      redirect: '/docs',
    };
  }
</script>
